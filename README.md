## Asri Theme Enhancement Plugin
An enhancement plugin designed specifically for the Asri theme.
### Update Notes
- New detail adjustment: Hide tabheader&breadcrumb
- New color scheme: Gruvbox
### More Preset Color Schemes
In addition to the original color schemes of the Asri theme, this plugin adds many preset color schemes
- Inspiration palette: Oxygen, Shade
- Ported from QYL theme: Amber, Wilderness, Midnight
- Ported from Savor theme: Salt
- Ported from Obsidian theme: Golden Topaz
- Ported from VSCode theme: Ros√© Pine
- Ported from Vim theme: Gruvbox
### Improved Editing Experience
Introduces commonly used features not yet supported by the Asri theme
- Arrow-key navigation in slash menu
- Typewriter mode
- List bullet line
### Theme Detail Adjustments
Detail adjustments based on the Asri theme's style
- Sidebar top sticky: Disables the Asri theme's sidebar top collapse/expand animation, keeping the top always visible
- Cover image fade: Adds a fade mask at the bottom of cover images
- Colored headings
- Colored file tree
- Hide tabheader&breadcrumb (only effective when top bar fusion+ is enabled, recommended to use with progressive blur effect)
