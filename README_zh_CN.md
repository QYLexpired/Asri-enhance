## Asri主题增强插件
专为Asri主题设计的增强插件
### 更新说明
- 新增细节调整：隐藏页签/面包屑
- 新增配色：格鲁夫（Gruvbox）
### 更多预设配色
在Asri主题原有配色基础上，额外增加大量预设配色方案
- 灵感配色：氧、林荫
- 移植于QYL主题：琥珀、旷野、午夜
- 移植于Savor主题：Salt
- 移植于Obsidian主题：Golden Topaz
- 移植于VSCode主题：Rosé Pine
- 移植于Vim主题：Gruvbox
### 提升编辑体验
引入Asri主题本体暂未支持的常用功能
- 斜杠菜单方向键导航
- 打字机模式
- 列表子弹线
### 主题细节调整
根据Asri主题的风格进行细节调整
- 侧栏顶部常驻显示：取消Asri主题的侧栏顶部收缩/展开动画，常驻显示顶部
- 题头图渐隐：为题头图底部增加渐隐遮罩
- 彩色标题
- 彩色文档树
- 隐藏页签/面包屑（仅在开启顶栏融合+时生效，建议搭配渐进模糊效果）
