### 更新
- 优化垂直页签，防止极个别情况下面包屑与顶栏按钮重合
- 为垂直页签栏增加垂直滚动时的溢出渐隐效果
- 修复开启故障配色时再开启其他配色不能持久化的问题
- 优化托帕石配色
- 优化大量细节
### 使用说明
- 点击顶栏<span data-type="kbd">外观模式</span>按钮，可出现Asri主题的设置菜单，本插件的所有功能在<span data-type="kbd">更多预设配色</span>和<span data-type="kbd">更多</span>两个选项中进行配置
- 本插件仅适配Asri主题
### 更多预设配色
在Asri主题原有配色基础上，额外增加大量预设配色方案
- 灵感配色：氧、林荫、故障
- 移植于QYL主题：琥珀、旷野、午夜
- 移植于Savor主题：Salt
- 移植于Obsidian主题：Golden Topaz
- 移植于VSCode主题：Rosé Pine
- 移植于Vim主题：Gruvbox
### 提升编辑体验
引入Asri主题本体暂未支持的实用功能
- 斜杠菜单方向键导航
- 打字机模式
- 列表子弹线
- 垂直页签（不能与顶栏融合+同时开启，只对左上角的页签栏生效）
### 主题细节调整
完全按照Asri主题的风格进行细节调整
- 侧栏顶部常驻显示：取消Asri主题的侧栏顶部收缩/展开动画，常驻显示顶部
- 题头图渐隐
- 纸张纹理
- 彩色标题
- 标题层级提示
- 彩色文档树
- 隐藏页签/面包屑（仅在开启顶栏融合+时生效，建议搭配渐进模糊效果）
