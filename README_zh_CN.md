## Asri主题增强插件

### 更新说明
- 增加细节调整功能
    - 侧栏顶部常驻显示：取消Asri主题的侧栏顶部收缩/展开动画，常驻显示顶部
    - 题头图渐隐：为题头图底部增加渐隐遮罩
- 移除彩色大纲开关（合并至彩色标题）
- 优化列表子弹线
